# Editing Core Skills

The six core skills (`/infil`, `/recon`, `/objs`, `/splash`, `/sitrep`, `/exfil`) are bundled into the `naholo` CLI. The CLI ships them into the user's `.claude/skills/` directory via `naholo skills install`.

- **Source of truth**: `packages/naholo-cli/src/skills/{name}.md` — edit these when changing skill prose.
- **Install target**: `.claude/skills/{name}/SKILL.md` — regenerated by `naholo skills install`; never edit these directly.

When the user asks to edit a core skill:

1. Edit `packages/naholo-cli/src/skills/{name}.md`.
2. Run `naholo skills install` to mirror the change into `.claude/skills/{name}/SKILL.md`.

Edits to `.claude/skills/{name}/SKILL.md` are lost the next time someone runs `naholo skills install`, so they don't survive.
